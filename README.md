# Crypto Market Snapshot Tool (CoinGecko API)

A lightweight Python script that fetches real-time cryptocurrency market data from the **CoinGecko API** and saves it into a clean, structured CSV file.  
This tool is perfect for automation, trend tracking, data analysis, and portfolio monitoring.

---

## 🚀 Features

- Fetches top cryptocurrencies by **market cap**
- Uses the public CoinGecko API (❌ no API key needed)
- Extracts important fields:
  - ID
  - Symbol
  - Name
  - Current Price
  - Market Cap
  - 24h Price Change %
  - Total Volume
- Automatically generates timestamped CSV filenames
- User-configurable coin count via CLI (`--per-page`)
- Simple, fast, and reliable
